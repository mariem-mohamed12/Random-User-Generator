# Random User Generator

A simple web application that generates random user data.

The app fetches random user information from an external API and displays user details on the screen.

## Features

- Generate random user on button click
- Fetch data from external API
- Display user image, name, age, and country
- Simple and clean UI

## Technologies Used

- HTML
- CSS
- JavaScript
- Bootstrap
- Random User API

## API Used

Random User API  
`https://randomuser.me/api`
